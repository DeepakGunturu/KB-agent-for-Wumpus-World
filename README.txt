To run this code:

1. Download all the files from the repository into a directory of your choice. Use terminal to go the directory where you have downloaded as follows: cd Desktop/directory-name
2. To play the game with a GUI, run it as follows: python3 wwsim.py -gui or python wwsim.py -gui, depending on the version of Python. This allows you to play the game as a user.
3. To run the KB agent without GUI to play the game, run the following command: ython3 wwsim.py -nongui or python wwsim.py -nongui, depending on the version of Python. This would enable the agent to play the game using an entailment.
